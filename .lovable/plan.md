
# אפליקציית סריקה למכשירי אנדרואיד - Alumkon Scanner

## גישה כללית

נבנה את האפליקציה כ**Web App מותאם מובייל** בתוך אותו פרויקט, במסלול נפרד `/scan`. זה יעבוד מצוין על מכשירי הסריקה האנדרואידיים (סורק ברקוד פיזי שמזין טקסט לשדה input + מצלמה דרך הדפדפן). בהמשך אפשר לעטוף ב-Capacitor ל-APK אמיתי אם תרצה - הקוד יישאר זהה.

**Backend**: נפעיל את Lovable Cloud כדי לשמור משתמשים אמיתיים, היסטוריית סריקות ותמונות (לא מספיק mock data לאפליקציה אמיתית בשטח).

## מבנה האפליקציה

מסלול נפרד `/scan` עם layout משלו (ללא sidebar, מותאם מסך קטן, אנכי, כפתורים גדולים לאצבעות עם כפפות).

```text
/scan/login         → התחברות (משתמש תחנה / משתמש בקר איכות)
/scan/station       → מסך מפעיל תחנה (רמה 1)
/scan/qc            → מסך בקר איכות (רמה 2)
```
